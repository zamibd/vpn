────────────────────────────────────────────────────────────────
📋 সব কমান্ড দ্রুত রেফারেন্স:
────────────────────────────────────────────────────────────────

শুরু করুন:
  docker compose up -d

লগ দেখুন:
  docker compose logs -f

স্থিতি যাচাই করুন:
  docker compose ps

পুনরায় শুরু করুন:
  docker compose restart

বন্ধ করুন:
  docker compose down

ভলিউম সহ মুছুন:
  docker compose down -v

Backend এ প্রবেশ করুন:
  docker compose exec backend sh

MySQL CLI:
  docker compose exec mysql mysql -u vpn_user -pvpn_password vpn_management

Backup নিন:
  docker compose exec mysql mysqldump -u vpn_user -pvpn_password vpn_management > backup.sql
